### v3.0.37（2026-07-04）

- **弱いモデル（DeepSeek等）の不正なツール呼び出しを正常に処理**。引数の型違いやJSONの切り詰めが発生してもエラーにならず実行を継続

### v3.0.35（2026-07-03）

- **ClinePassを全CLIユーザーに有効化**
- Tencent TokenHubをプロバイダーとして追加（SDK v0.0.55から）
- 高出力モデル（MiniMax M3等）での最初のプロンプト切り詰めを修正。即座に自動コンパクトされて初期タスクが入力ラッパーのみになる問題を解決

### AI・GitHub Copilot関連の変更（抜粋）

#### エージェント機能
- **ブラウザツールがGA**: エージェントが統合ブラウザでページを開き、コンテンツやコンソールエラーを読み取り、スクリーンショットを撮って、選択・入力・ナビゲーションで自身の作業を検証可能（デフォルト有効、外部MCPサーバー不要）
- **エージェントウィンドウ（プレビュー）**: セッションリストをグループ化して整理、ドラッグ&ドロップで並べ替え、ピン留め
- **チャット入力バナー**: CI失敗やPRレビューコメントを会話から離れずに対応。「Fix Checks」「Address Comments」アクション
- **サブエージェントクレジット**: サブエージェントセクションをホバーすると使用AIクレジットを表示
- **エディターガターからのフィードバック**: 行をホバーすると「Add Feedback」グリフが表示され、特定行にコメントを追加可能

#### マルチチャットセッション改善
- チャットの閉じる・再開・削除、進捗と変更の全チャット横集約、会話のピアチャットへのフォーク

#### ターミナルサンドボックス（macOS/Linux）
- エージェント実行のターミナルコマンドをサンドボックス化。ネットワーク遮断・ファイルシステム制限付きで実行し、承認プロンプトを削減

#### モデル・エンタープライズ
- **組み込みOllamaプロバイダーを非推奨化**。公式Ollama拡張機能への移行を推奨
- ファイルベースの管理Copilot設定配信（`managed-settings.json`）を追加。MDM未登録マシンでのポリシー適用が容易に

## Cursor

- **ウェブサイト**: https://cursor.com
- **チェンジログ**: https://cursor.com/changelog

### 最新の変更点

#### Cursor iOS版（公開ベータ）
- 全有料プランでモバイルアプリの公開ベータを開放。常駐エージェントの起動・管理が可能
- モバイルからのクラウドエージェント起動、音声入力、スラッシュコマンド対応
- リモートコントロール: デスクトップのエージェントをスマートフォンに切り替えて操作
- リアルタイムアクティビティとプッシュ通知でエージェント状態を追跡

#### カスタマイズ機能
- プラグイン、スキル、MCP、サブエージェント、ルール、コマンド、フックを一元管理する「Customize」ページ
- チームマーケットプレイスランキング: チーム内で人気のプラグイン・スキル・MCPを表示
- プラグインキャンバス: チームで共有・再利用可能なセットアップテンプレート（Hex、Atlassian等）

#### チームマーケットプレイス拡張
- チームMCPと組織グループをサポート。管理者が一度設定するだけでクラウドエージェント、エージェントウィンドウ、IDE、CLI全体に配信
- GitLab、BitBucket、Azure DevOpsからのプラグインリポジトリインポートに対応

#### Cursor Automations改善
- **`/automate`スキル**: ローカルエージェントセッション内で自然言語による自動化作成
- **Slack絵文字トリガー**: 指定した絵文字リアクションで自動化を起動
- **GitHubトリガー拡張**: Issue comment、PRレビューコメント、PR review submitted、Review thread updated、Workflow run completedの5種を追加
- コンピュータ操作ツール: 自動化から起動されたクラウドエージェントが独立したコンピュータで成果物のデモを生成可能

#### クラウド環境設定・クラウドサブエージェント
- 10分未満でクラウド開発環境をセットアップ。環境を再利用可能なスナップショットとして保存
- **`/in-cloud`**: 独立VMで動くクラウドサブエージェントを起動。長時間実行や並列作業を分離し、ローカルワークスペースをクリーンに保つ
- ローカルとクラウド間でエージェントセッションをより確実に切り替え

### IDE 1.0.89（2026-07-03）

- **セッションリストア**: 起動時にセッションが自動的に復元
- パフォーマンス最適化とAgent Focus Modeの改良

- 権限、フック、ステアリングがウィンドウやカスタムエージェントプロファイル間で正しく動作するよう複数の修正
- ※最新1.0.xリリース（1.0.89以降）は kiro.dev/downloads から直接ダウンロード。自動更新は一時停止中

### CLI 2.11.0（2026-07-02）

- **MCP認証管理**: リモートMCPサーバーのOAuth認証を管理する専用コマンドを追加
  - `/mcp auth`: トークン期限切れや無効時に再認証を強制
  - `/mcp cancel-auth`: ブラウザ確認待ちでスタックした認証フローを中止
  - `/mcp logout`: 保存された認証情報を削除
  - キーボードショートカット: `^A`（強制認証）、`^X`（中止）、`^R`（資格情報削除）
- `/usage`表示を刷新し、新しいプリペイドクレジットモデルに対応

### 全般アップデート（2026-07-01 / 07-02）

- **カスタム超過キャップ（7月2日）**: エンタープライズ管理者がAWS Service Quotasでユーザーごとの超過上限を設定可能
- **個人プラン向けアドオンクレジット（7月1日）**: Pro、Pro+、Pro Max、Powerプランで月額割り当て消費後にプリペイドクレジットを購入可能（$0.04/クレジット、$5〜$100のパック、12ヶ月有効）
- **Claude Sonnet 5提供開始（7月1日）**: Kiro IDE、CLI、Webで利用可能。Sonnet 4.6から推論・ツール使用・コーディング能力を大幅強化。1Mコンテキストウィンドウ、クレジット倍率1.3x
- **Web向けIAMロールとサンドボックス（7月1日）**: タスク実行時にKiro Webがユーザーに代わってAWSリソースにアクセス。Figma、Stripe、Supabase等のサードパーティサービス向けの標準認証フローを追加
