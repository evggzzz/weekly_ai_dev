### 🎬 OpenMontage: AIコーディングアシスタントを動画制作スタジオに変える

[calesthio/OpenMontage](https://github.com/calesthio/OpenMontage)

世界初のオープンソース・エージェント動画制作システム。AIコーディングアシスタントに自然言語で指示するだけで、企画・脚本・アセット生成・編集・最終構成までを自動で処理してくれる。

他のAI動画ツールが「プロンプトから1つのクリップ」を生成するのとは違う。OpenMontageは本物の制作パイプラインを提供する。12のプロダクションパイプライン（アニメーション解説、トーキングヘッド、画面デモ、シネマティックトレーラー、アニメーション、ポッドキャスト、ローカライズ、ドキュメンタリーモンタージュなど）、52の制作ツール、500以上のエージェントスキルを備えている。

**最大の特徴は「参照動画駆動」**: 好きな動画（YouTube/Short/Reel/TikTok）を貼り付けるだけで、エージェントがトランスクリプト、ペーシング、シーン、キーフレーム、スタイルを分析し、差別化されたコンセプトを2〜3個提案してくれる。しかも、コスト見積もりとサンプル付きだ。

**APIキーなしでも動く**: Piper TTSでナレーション、Archive.org/NASA/Wikimedia Commonsから無料アーカイブ映像、Pexels/Unsplash/Pixabayから無料ストックフッテージ、Remotionでアニメーション、FFmpegでポストプロダクション、WhisperXで字幕生成。ゼロコストからスタートできる。

**GPUを持っていれば無料ローカル動画生成も可能**: WAN 2.1、Hunyuan、CogVideo、LTX-Videoなどのローカルモデルをサポート。

制作ガバナンスも充実している。プリコンポーズ検証で「スライドショー化」をブロック、ポストレンダ自己レビューでffprobe検証、フレーム抽出、音声レベル分析、配達約束検証、字幕チェックを実行。プロバイダー選択は7次元（タスク適合性、出力品質、制御性、信頼性、コスト効率、レイテンシ、継続性）でスコアリングされ、監査可能な決定ログに記録される。

**料金管理も組み込み済み**: 実行前の見積もり、予確保、実行後の調整、設定可能なモード、アクションごとの承認しきい値、総予算上限。驚きの請求はない。

AIアシスタント（Claude Code/Cursor/Copilot/Windsurf/Codex）を持っていればすぐ使える。`git clone`して`make setup`するだけ。「60秒のニューラルネットワーク学習アニメーション解説を作って」と言えば、エージェントがWeb検索、スクリプト、音声、フッテージ、音楽、字幕、レンダリングまで全部やってくれる。

### ☁️ AWS公式のAIエージェントツールキット: エージェントがAWSで構築・デプロイ・管理するための公式サポート

[aws/agent-toolkit-for-aws](https://github.com/aws/agent-toolkit-for-aws)

AWSが公式で提供するAIコーディングエージェント用ツールキット。MCPサーバー、スキル、プラグインを通じて、AIエージェントがAWSサービスを使って構築・デプロイ・管理できるようにする。

Claude Code、Codex、Cursor、Kiroなど、開発者が既に使っているコーディングエージェントで動作する。

**含まれるもの**:
- **MCPサーバー**: 300以上のAWSサービスを単一の認証エンドポイントから操作可能。サンドボックス化されたスクリプト実行、リアルタイムドキュメントアクセス、CloudWatchメトリクス、IAMコンテキストキー、CloudTrail監査ログ
- **プラグイン**: AWS MCPサーバー設定とエージェントスキルを1つにまとめたインストールパッケージ
  - `aws-core`: サービス選択、CDK/CloudFormation、サーバーレス、コンテナ、ストレージ、可観測性、請求、SDK使用、デプロイメント
  - `aws-agents`: Amazon BedrockとAgentCoreでAWS上にAIエージェントを構築
  - `aws-data-analytics`: S3 Tables、AWS Glue、Athenaでのデータレイク、アナリティクス、ETLワークフロー
  - `aws-agents-for-devsecops`: インシデント調査、コードレビューとUAT実行、脆弱性スキャン、ペネトレーションテスト（AWS DevOps AgentとAWS Security Agentを使用）
- **スキル**: 特定のAWSタスクを完了するための指示と参照資料のキュレーションパッケージ。オンデマンドでロードされる
- **ルールファイル**: エージェントがAWSを最も効果的に使う方法を教えるプロジェクトレベル設定ファイル

**Claude Codeでのクイックスタート**:

```
/plugin install aws-core@claude-plugins-official
```

AWS LabsのMCPサーバー、スキル、プラグインの後継。IAM条件キーでエージェントアクションとヒューマンアクションを区別してポリシー適用、CloudWatchメトリクスとCloudTrail監査ログ、エンドツーエンド評価を受けたエージェントスキルなど、主要な機能が強化されている。

**エージェント専用のIAM条件キーが重要**: エージェントのアクションにのみ適用するポリシーが書ける。例えば、ユーザーのIAMロールが書き込み可能でも、MCPサーバー経由では読み取り専用に制限できる。

ドキュメントも充実しており、ユーザーガイド（セットアップ、設定、リファレンスドキュメント）とAWS MCPサーバーツール（利用可能なMCPツールのリファレンス）が用意されている。

AIエージェントがAWSを扱うための公式サポートが整ったことで、エージェント駆動のインフラ構築が本格的に実用的になっている。
