### OpenMontage: AIコーディングアシスタントを動画制作スタジオに変える

https://github.com/calesthio/OpenMontage

オープンソース・エージェント動画制作システム。AIコーディングアシスタントに自然言語で指示するだけで、企画・脚本・アセット生成・編集・最終構成までを自動処理します。

他のAI動画ツールが「プロンプトから1つのクリップ」を生成するのとは異なり、本物の制作パイプラインを提供します。12のプロダクションパイプライン（アニメーション解説、トーキングヘッド、画面デモ、シネマティックトレーラー、ポッドキャスト、ローカライズ、ドキュメンタリーモンタージュなど）、52の制作ツール、500以上のエージェントスキルを備えています。

最大の特徴は「参照動画駆動」です。好きな動画（YouTube/Short/Reel/TikTok）を貼り付けるだけで、エージェントがトランスクリプト、ペーシング、シーン、キーフレーム、スタイルを分析し、差別化されたコンセプトを2〜3個提案します。コスト見積もりとサンプル付きです。

APIキーなしでも動作します。Piper TTSでナレーション、Archive.org/NASA/Wikimedia Commonsから無料アーカイブ映像、Pexels/Unsplash/Pixabayから無料ストックフッテージ、Remotionでアニメーション、FFmpegでポストプロダクション、WhisperXで字幕生成。GPUがあればWAN 2.1、Hunyuan、CogVideo、LTX-Videoなどのローカルモデルで無料動画生成も可能です。

制作ガバナンスも充実しています。プリコンポーズ検証で「スライドショー化」をブロック、ポストレンダ自己レビューでffprobe検証・フレーム抽出・音声レベル分析・字幕チェックを実行。プロバイダー選択は7次元（タスク適合性、出力品質、制御性、信頼性、コスト効率、レイテンシ、継続性）でスコアリングされ、監査可能な決定ログに記録されます。実行前見積もり・予算確保・アクションごとの承認しきい値・総予算上限による料金管理も組み込み済みです。

### AWS公式AIエージェントツールキット

https://github.com/aws/agent-toolkit-for-aws

AWSが公式で提供するAIコーディングエージェント用ツールキット。MCPサーバー、スキル、プラグインを通じて、AIエージェントがAWSサービスを使って構築・デプロイ・管理できるようにします。Claude Code、Codex、Cursor、Kiroなど、開発者が既に使っているコーディングエージェントで動作します。

含まれるもの:
- **MCPサーバー**: 300以上のAWSサービスを単一の認証エンドポイントから操作可能。サンドボックス化されたスクリプト実行、リアルタイムドキュメントアクセス、CloudWatchメトリクス、IAMコンテキストキー、CloudTrail監査ログ
- **プラグイン**: AWS MCPサーバー設定とエージェントスキルを1つにまとめたインストールパッケージ
  - `aws-core`: サービス選択、CDK/CloudFormation、サーバーレス、コンテナ、ストレージ、可観測性、請求、SDK使用、デプロイメント
  - `aws-agents`: Amazon BedrockとAgentCoreでAWS上にAIエージェントを構築
  - `aws-data-analytics`: S3 Tables、AWS Glue、Athenaでのデータレイク、アナリティクス、ETLワークフロー
  - `aws-agents-for-devsecops`: インシデント調査、コードレビューとUAT実行、脆弱性スキャン、ペネトレーションテスト
- **スキル**: 特定のAWSタスクを完了するための指示と参照資料のキュレーションパッケージ。オンデマンドでロードされる
- **ルールファイル**: エージェントがAWSを最も効果的に使う方法を教えるプロジェクトレベル設定ファイル

Claude Codeでのクイックスタート:

```
/plugin install aws-core@claude-plugins-official
```

エージェント専用のIAM条件キーが重要です。エージェントのアクションにのみ適用するポリシーが書けます。例えば、ユーザーのIAMロールが書き込み可能でも、MCPサーバー経由では読み取り専用に制限できます。AWS LabsのMCPサーバー・スキル・プラグインの後継であり、エージェント駆動のインフラ構築が本格的に実用的になっています。
